# Planificator listă de cumpărături

Aplicație web simplă care permite:

- lipirea rapidă a unei liste de produse (copy/paste);
- selectarea supermarket-ului (Lidl, Kaufland, Carrefour);
- ordonarea listei în funcție de traseul tipic prin magazin.

## Rulare locală

Deschide `index.html` în browser sau rulează un server static:

```bash
python3 -m http.server 4173
```

Apoi accesează `http://localhost:4173`.
